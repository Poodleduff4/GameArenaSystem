Instructions to run Ticket Sensei

The only additional requirements outside of Java is Python3 and some small packages

If the machine doesn't have python3 already installed, run this command:
sudo apt-get install python3.6

once this is finished, install the packages listed below:
Pillow
qrcode

by using the command:
python3 -m pip install Pillow qrcode

Once these are done installing, you are ready to run the application
